# SOCloom Brand Assets

Minimal dark-theme assets for GitHub README and documentation.

## Files

| File | Use |
|------|-----|
| `logo.svg` | Primary logo (vector, scalable) |
| `logo-mark.svg` | Icon / favicon source |
| `logo.png` | Raster logo for social / OG |
| `banner.svg` | GitHub README hero banner |
| `banner.png` | Raster banner for social sharing |
| `social-preview.svg` | GitHub social preview (1280×640) |
| `charts/detection-layers.svg` | 3-layer detection diagram |
| `charts/skills-distribution.svg` | Skills catalog bar chart |
| `charts/risk-analytics.svg` | Dashboard analytics preview |
| `charts/architecture-flow.svg` | Pipeline flow diagram |

## Colors

| Name | Hex | Usage |
|------|-----|-------|
| Deep Navy | `#060d18` | Background |
| Panel | `#0b1220` | Cards, charts |
| Cyan | `#06b6d4` | Primary accent |
| Teal | `#22d3ee` | Secondary accent |
| Purple | `#8b5cf6` | Rules / skills accent |
| Green | `#10b981` | Success / low risk |

## Logo Concept

Hexagonal shield (security) with three interwoven threads (loom) representing ML, rules, and skill-orchestrated AI woven together.
